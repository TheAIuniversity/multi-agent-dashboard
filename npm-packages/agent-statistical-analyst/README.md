# Statistical Analysis Expert

Advanced statistical modeling, regression analysis, and predictive analytics

## Installation

```bash
npx @claude-code/agent-statistical-analyst
```

## About

This agent specializes in statistical analysis expert tasks and will be automatically triggered by Claude Code when relevant work is detected.

## Features

- Automatic delegation from Claude Code
- Specialized expertise in statistical analyst
- Integrated with Multi-Agent Dashboard for monitoring

## Usage

Once installed, this agent will be automatically used by Claude Code when appropriate. No manual configuration required.

## Dashboard Integration

To monitor this agent's activity in real-time:

```bash
npx multi-agent-dashboard-connect
```

Then open http://localhost:5174 in your browser.
